# Donoji

Gem to help out finding character and word uses from mainichi shinbun (japanese newspaper)

## Prerequisites

Make sure `bundler` is installed. If not run `gem install bundler`

## Installation

    $ bundle install
    $ rake build
    $ gem install pkg/donoji-0.2.0.gem

## Usage

    $ donoji # CLI
    $ donoji find é›… -v
